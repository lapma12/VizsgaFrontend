# Vizsga Frontend

Rövid mappastruktúra:
    - `src/Pages` - oldal komponensek <br/>
    - `src/Pages/Services` - szolgáltatás oldalak 
    - `src/Components` - több oldalon használt közös JSX komponensek 
    - `src/Styles` - CSS fájlok oldalakhoz és közös elemekhez
    - `src/Img` - képek (logó, háttérképek, illusztrációk)
    - `src/assets` - egyéb statikus fájlok (pl. videó, letölthető játék fájl)
    - `src/api` - API kliens
    - `src/Tests` + `setupTests.js` - frontend tesztek

Belépési pontok:
    - `src/main.jsx`
    - `src/App.jsx`
